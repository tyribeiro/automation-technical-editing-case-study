# Assessment-Style Questions

The following questions demonstrate how workflow knowledge can be converted into certification-style assessment items.

---

## Question 1
What is the primary purpose of checking the Google Sheet before continuing the scenario?

A. To reduce data storage costs  
B. To prevent duplicate processing  
C. To validate ClickUp authentication  
D. To refresh Smartsheet values  

Correct answer: B

---

## Question 2
Which step must occur before updating Smartsheet?

A. Deploying the scenario  
B. Retrieving the sheet using `smartsheet_id`  
C. Resetting workflow variables  
D. Adding a row to ClickUp  

Correct answer: B

---

## Question 3
Why must numeric column IDs be used when updating Smartsheet?

A. They are required for API authentication  
B. They ensure consistent mapping even if column names change  
C. They reduce execution time  
D. They support larger payloads  

Correct answer: B

---

## Question 4
What should happen when the `list_id` already exists in the Google Sheet?

A. Skip only the Smartsheet step  
B. Terminate the scenario  
C. Delete the row from Google Sheets  
D. Retry the scenario three times  

Correct answer: B
